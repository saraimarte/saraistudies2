U-Substitution is the *first* strategy you need to use when evaluating integrals if using power rule does not work.

&nbsp;&nbsp;&nbsp;&nbsp;

U-Sub takes takes the hard looking parts of an integral and replaces them with U so that it's easier to integrate.

&nbsp;&nbsp;&nbsp;&nbsp;

I came up with CUP- an acronym to help you remember the steps.  

&nbsp;&nbsp;&nbsp;&nbsp;

C- What we **C**are about.  
U- Put everything in terms of **U**  
P- **P**lug U back in  

&nbsp;&nbsp;&nbsp;&nbsp;

See how I implement CUP in the following examples,
