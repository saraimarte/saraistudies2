## What is a series

&nbsp;&nbsp;&nbsp;&nbsp;

A series is just the sum of a sequence.

&nbsp;&nbsp;&nbsp;&nbsp;

In other words, you are adding up all the terms in a sequence.

&nbsp;&nbsp;&nbsp;&nbsp;

## Notation

&nbsp;&nbsp;&nbsp;&nbsp;

So, if a sequence is: &nbsp;&nbsp;&nbsp;&nbsp;  a<sub>n</sub> = ``` {1, 2, 3...} ```  

&nbsp;&nbsp;&nbsp;&nbsp;

To find the series you take the partial sum at every n and dump that into the sequence of partial sums called s<sub>n</sub>.

&nbsp;&nbsp;&nbsp;&nbsp;

It's called partial sums because you will notice we take the sum one n at a time- partially.

&nbsp;&nbsp;&nbsp;&nbsp;

s<sub>1</sub> = 1 = 1 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; so, &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; S<sub>n </sub> = ```{1, ...} ```  
s<sub>2</sub> = 1 + 2 = 3 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;  so, &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;S<sub>n </sub> = ```{1, 3, ...} ```  
s<sub>3</sub> = 1 + 2 + 3 = 6 &nbsp;&nbsp;&nbsp;&nbsp;  &nbsp;&nbsp;&nbsp;&nbsp;  &nbsp;so, &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; S<sub>n </sub> = ```{1, 3, 6 ...} ``` 


&nbsp;&nbsp;&nbsp;&nbsp;

then the series is:  &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp; s<sub>n</sub> = ``` {1 + 3 + 6...} ```  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;   &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;s<sub>1</sub>&nbsp;&nbsp; s<sub>2</sub> &nbsp;&nbsp;  s<sub>3</sub>

&nbsp;&nbsp;&nbsp;&nbsp;

So, if I ask you: **What is S<sub>3</sub>?** You say 6 because (1 + 2 + 3) = 6

&nbsp;&nbsp;&nbsp;&nbsp;

I'm really asking what is the partial sum of a<sub>n</sub> up until n = 3.

&nbsp;&nbsp;&nbsp;&nbsp;

In sigma notation, I'm really finding:

## How do we find the sum of the *whole* sequence?

&nbsp;&nbsp;&nbsp;&nbsp;

But if the goal is to find the full sum of the sequence a<sub>n</sub> then that's going to take forever. Because a sequence will go on forever and so will I have to sit here and find the partial sums for forever. Imagine if I had to find S<sub>100000</sub>, we don't have all day. So what do we do?

&nbsp;&nbsp;&nbsp;&nbsp;

We literally take the sum of the sequence from a starting point to infinity.

&nbsp;&nbsp;&nbsp;&nbsp;

<div style = "display:flex; flex-direction:column; gap: 20px; flex-wrap:wrap" >

So, instead of finding the sum of the sequence from one point to another finite point n, 

<div style = "object-fit:contain;" > <img src={img1.src} alt=""  /> </div>

We can find the sum of the sequence from one point to infinity. 

<div style = "object-fit:contain;" > <img src={img3.src} alt=""  /> </div>

</div>

&nbsp;&nbsp;&nbsp;&nbsp;

but remember infinity is not a number, so really we are trying to find the sum of the sequence as it approaches infinity.

&nbsp;&nbsp;&nbsp;&nbsp;

<div style = "object-fit:contain;" > <img src={img4.src} alt=""  /> </div>

&nbsp;&nbsp;&nbsp;&nbsp;

which is really,

&nbsp;&nbsp;&nbsp;&nbsp;

<div style = "object-fit:contain;" > <img src={img25.src} alt=""  /> </div>

&nbsp;&nbsp;&nbsp;&nbsp;

which is the sum of the sequence of partial sums as n gets infinitely bigger aka the series. 

&nbsp;&nbsp;&nbsp;&nbsp;

So, if the sequence of partial sum (S<sub>n</sub>) converges then the series converges (because the series is literally the sum of the sequence of partial sums as n goes to infinity).

&nbsp;&nbsp;&nbsp;&nbsp;

This tells us that to find if a series converges we really just need to find if the sequence of partial sums converges.

&nbsp;&nbsp;&nbsp;&nbsp;

## What do we care about?

&nbsp;&nbsp;&nbsp;&nbsp;

As the last lesson, there are only three things we care about, if a series converges, diverges and if it converges what does it converge to?

&nbsp;&nbsp;&nbsp;&nbsp;

### How do we know when a series converges or diverges?

&nbsp;&nbsp;&nbsp;&nbsp;

First, we have three special types of series that when we know something about them, we can conclude they converge.

&nbsp;&nbsp;&nbsp;&nbsp;

Those are geometric, telescoping and p-series or harmonic series. Those we will go over first.

&nbsp;&nbsp;&nbsp;&nbsp;

Second, we will then move onto tests. These tests will tell us if a series converge or not.

&nbsp;&nbsp;&nbsp;&nbsp;

### How do we know what a series converges to, if it converges?

&nbsp;&nbsp;&nbsp;&nbsp;

We will only be able to find out what a series converges to if we're dealing with telescoping or geometric series (10.2) because either we can find it manually or we have a formula for the sum. After 10.2, we will only be able to tell if the series converges or not.

&nbsp;&nbsp;&nbsp;&nbsp;

## 10.2 Geometric Series

&nbsp;&nbsp;&nbsp;&nbsp;

Series where every term contains a common ratio r.

&nbsp;&nbsp;&nbsp;&nbsp;

So s<sub>n</sub> = ```{2 + 4 + 8   + ...}```

&nbsp;&nbsp;&nbsp;&nbsp;

is really, s<sub>n</sub> = ```{a + ar + ar^2 + ...}```

&nbsp;&nbsp;&nbsp;&nbsp;

Note: most of the time you won't be given the series in just a list with dots ```{...}``` instead it will be given to you in sigma notation or closed form like so,

&nbsp;&nbsp;&nbsp;&nbsp;

<div style = "object-fit:contain;" > <img src={img22.src} alt=""  /> </div>

&nbsp;&nbsp;&nbsp;&nbsp;

Here, a = 2 r = 2   

&nbsp;&nbsp;&nbsp;&nbsp;

a is the first term of the series.   
r is the second term divided by the first (4/2 = 2)

&nbsp;&nbsp;&nbsp;&nbsp;

Just by knowing what r is we can find out if the series converges.

&nbsp;&nbsp;&nbsp;&nbsp;

Rule: 

If |r| < 1 CON  
If |r| ≥ 1 DIV  

&nbsp;&nbsp;&nbsp;&nbsp;

Here, because |2| ≥ 1, the series S<sub>n</sub> diverges.

## 10.5 Absolute Convergence Tests

&nbsp;&nbsp;&nbsp;&nbsp;

What happens when we have series with negative terms?

&nbsp;&nbsp;&nbsp;&nbsp;

Well, because mathematicians are lazy they just said "hey! we don't have to work with series with negative terms- we can just put absolute value signs around them and boom! now they're positive series".

&nbsp;&nbsp;&nbsp;&nbsp;

But doesn't that make it a different series?

&nbsp;&nbsp;&nbsp;&nbsp;

Yes! But the idea is that when you add absolute value to something the worst you can do is make that something bigger. So the concept is that if the series with absolute value converges then so will the series without absolute value (the given series) because that series will always be smaller or equal to it's corresponding positive series. So, if the top series converges the bottom will have to follow the same behavior (Think DCT).

&nbsp;&nbsp;&nbsp;&nbsp;

So, now we use absolute convergence tests that tell us if a series with absolute value converges or not.

&nbsp;&nbsp;&nbsp;&nbsp;

If it does, we say it converges absolutely (because it literally has absolute value symbols around it).

&nbsp;&nbsp;&nbsp;&nbsp;
