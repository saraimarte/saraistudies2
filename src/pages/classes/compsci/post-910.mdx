**What you know:** arrays can hold any data type, both primitive (ints, doubles, chars) or abstract (Strings).

&nbsp;&nbsp;&nbsp;&nbsp;

**What's new:** array lists can hold *only* objects.

&nbsp;&nbsp;&nbsp;&nbsp;

You're going to love these because this is how you wished arrays worked.

&nbsp;&nbsp;&nbsp;&nbsp;

Let's start.

&nbsp;&nbsp;&nbsp;&nbsp;

First, make sure you import the class at the top of your file.

&nbsp;&nbsp;&nbsp;&nbsp;


```java

    import java.util.ArrayList;


```

&nbsp;&nbsp;&nbsp;&nbsp;

Why? Because ArrayList is not built into java like String is. ArrayList is like the Color class/object where you had to import it.

&nbsp;&nbsp;&nbsp;&nbsp;

## How to declare and initialize an array list?

&nbsp;&nbsp;&nbsp;&nbsp;

Let's declare and initialize an arrayList that contains only String items.

&nbsp;&nbsp;&nbsp;&nbsp;

```java

    ArrayList <String> names = new ArrayList <String> ();


```

&nbsp;&nbsp;&nbsp;&nbsp;

Always follow the same format: 

&nbsp;&nbsp;&nbsp;&nbsp;

```java

    ArrayList <datatypeOfItems> arrayListName = new ArrayList <datatypeOfItems> ();


```

&nbsp;&nbsp;&nbsp;&nbsp;

Note: ArrayLists can only hold object of the same type. So Only Strings not Colors and Strings or Pictures and Strings.

&nbsp;&nbsp;&nbsp;&nbsp;

## ArrayList Methods

&nbsp;&nbsp;&nbsp;&nbsp;

Just like every other object- ArrayList can and does have some methods we can use specifically on this object.

&nbsp;&nbsp;&nbsp;&nbsp;

## **1.ArrayList.size()**

&nbsp;&nbsp;&nbsp;&nbsp;

Returns the amount of items in the arrayList.

&nbsp;&nbsp;&nbsp;&nbsp;


```java


    ArrayList <String> names = new ArrayList <String> ();
    
    StdOut.println(names.size());

    //Output: 0;


```

&nbsp;&nbsp;&nbsp;&nbsp;

Right now, size is 0 because there is nothing in the array but we can add things...

&nbsp;&nbsp;&nbsp;&nbsp;

## **2.ArrayList.add(x)**

&nbsp;&nbsp;&nbsp;&nbsp;

Adds x to the end of the arrayList.

&nbsp;&nbsp;&nbsp;&nbsp;


```java


    ArrayList <String> cars = new ArrayList <String> ();
    
    names.add("Sarai"); //adds sarai to the list of names

    StdOut.println(names); //ISNT THIS AMAZING THAT YOU DONT HAVE TO LOOP THROUGH IT TO PRINT OMGGG
    StdOut.println(names.size()); //print amount of items in the arrayList

    //Output: [Sarai]
    //Output: 1


```

&nbsp;&nbsp;&nbsp;&nbsp;

## **3.ArrayList.add(n,object)**

&nbsp;&nbsp;&nbsp;&nbsp;

Adds object to the nth position in the array list.

&nbsp;&nbsp;&nbsp;&nbsp;


```java

   
    ArrayList <String> names = new ArrayList <String> ();

    //add "Sarai" at the 0th index 
    //aka add "Sarai" as the first item in the array list
  
    names.add(0, "Sarai");
  
    StdOut.println(names); //Output: [Sarai]
    StdOut.println(names.size()); //Output: 1
   


```

&nbsp;&nbsp;&nbsp;&nbsp;

## **4.ArrayList.remove(index)**

&nbsp;&nbsp;&nbsp;&nbsp;

Removes object at specified index.

&nbsp;&nbsp;&nbsp;&nbsp;


```java


    ArrayList <String> names = new ArrayList <String> ();
    names.add("Sarai");
    
    //removes object at 0th index 
    //aka removes the first item in the array list.
    names.remove(0); 

    StdOut.println(names); //Output: []
    StdOut.println(names.size()); //Output: 0


```

&nbsp;&nbsp;&nbsp;&nbsp;

## **5.ArrayList.remove(object)**

&nbsp;&nbsp;&nbsp;&nbsp;

Removes specified object.

&nbsp;&nbsp;&nbsp;&nbsp;


```java
   

    ArrayList <String> names = new ArrayList <String> ();
    names.add("Sarai");
    
    //removes "Sarai" from the arrayList
    names.remove("Sarai"); 

    StdOut.println(names); //Output: []
    StdOut.println(names.size()); //Output: 0
   


```

&nbsp;&nbsp;&nbsp;&nbsp;

## **6.ArrayList.indexOf(object)**

&nbsp;&nbsp;&nbsp;&nbsp;

Returns the position of an object in the array list.

&nbsp;&nbsp;&nbsp;&nbsp;


```java
   
    ArrayList <String> names = new ArrayList <String> ();
    
    names.add("Sarai");
   
    StdOut.println(names.indexOf("Sarai")); 
    
    //Output: 0


```


&nbsp;&nbsp;&nbsp;&nbsp;

## **7.ArrayList.get(object)**

&nbsp;&nbsp;&nbsp;&nbsp;

Gets the object at specified index.

&nbsp;&nbsp;&nbsp;&nbsp;


```java

       
    ArrayList <String> names = new ArrayList <String> ();
    
    names.add("Nikki");
    names.add("Hiro");
    names.add("Claire");
    names.add("Peter");

    StdOut.println(names.get(2));
    
    //Output: Claire

   
```


&nbsp;&nbsp;&nbsp;&nbsp;

## **8.ArrayList.set(object)**

&nbsp;&nbsp;&nbsp;&nbsp;

Replaces the object at specified index.

&nbsp;&nbsp;&nbsp;&nbsp;


```java
  
         
    ArrayList <String> names = new ArrayList <String> ();
    
    names.add("Nikki");
    names.add("Hiro");
    names.add("Claire");
    names.add("Peter");

    //Before: [Nikki, Hiro, Claire, Peter]

    //Replace the first item in the array list (0th index) with Sylar
    names.set(0, "Sylar");

    StdOut.println(names);
    
    //Output: [Sylar, Hiro, Claire, Peter]
   

```
