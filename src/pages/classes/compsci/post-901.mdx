## 2 Dimensional Arrays

&nbsp;&nbsp;&nbsp;&nbsp;

We can also make 2D arrays- that simply means we now have rows and columns.

&nbsp;&nbsp;&nbsp;&nbsp;

```java

    int[][] numbers = new int[2][2];

    for(int i = 0; i < numbers.length; i++ ){
        System.out.println(numbers[i][i]);
    }

    //Output: 


```

&nbsp;&nbsp;&nbsp;&nbsp;

```java

    int[][] numbers = new int[10][5];

    for (int row = 0; row < numbers.length; row ++) {
        for(int column = 0; column < numbers.length; column++){
            System.out.println(numbers[row][column]);
        }
    }

    //Output: 


```

## Initializing and Declaring

&nbsp;&nbsp;&nbsp;&nbsp;

Instead of int[] we do int[][]

&nbsp;&nbsp;&nbsp;&nbsp;

int[][] arr = new int[2][2]

&nbsp;&nbsp;&nbsp;&nbsp;

We call this a two-by-two array, meaning two rows and two columns.

&nbsp;&nbsp;&nbsp;&nbsp;

Again, by default, Java initializes the arrays to 0.

&nbsp;&nbsp;&nbsp;&nbsp;

So, right now, we have 

&nbsp;&nbsp;&nbsp;&nbsp;

0 0  
0 0  

&nbsp;&nbsp;&nbsp;&nbsp;

We can change those values manually.

&nbsp;&nbsp;&nbsp;&nbsp;

arr[row][col]

&nbsp;&nbsp;&nbsp;&nbsp;

arr[0][0] = 1;  
arr[0][1] = 2;  
arr[1][0] = 3;  
arr[1][1] = 4;  

&nbsp;&nbsp;&nbsp;&nbsp;

Let's also cover how to change them automatically because what if you have an array of size 100- you won't manually adjust each value. Before that, we have to learn how to go through a 2D array- let's do that and print every row and column of numbers given the variety we just had.

&nbsp;&nbsp;&nbsp;&nbsp;

```java

    public class fileName {
        
        public static void main(String[] args){

            int[][] arr = new int[2][2];

            //row 0 column 0
            arr[0][0] = 1;
            arr[0][1] = 2;
            arr[1][0] = 3;
            arr[1][1] = 4;

            for (int row = 0; row < arr.length; row++){ //for every row
                //go through every column
                StdOut.println(" ");
                for (int col = 0; col < arr.length; col++){
                    StdOut.print(arr[row][col] + " ");
                }
            }

        }

    }

    //Output 
  //  1 2 
   // 3 4 


```
