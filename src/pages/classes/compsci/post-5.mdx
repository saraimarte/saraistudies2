## Math Library 

&nbsp;&nbsp;&nbsp;&nbsp;

### Introduction

&nbsp;&nbsp;&nbsp;&nbsp;

Have you ever asked yourself how to square a number in java?

&nbsp;&nbsp;&nbsp;&nbsp;

Well, you can multiply it by itself...

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   System.out.println(4*4); //same thing as 4^2 = 16


```
&nbsp;&nbsp;&nbsp;&nbsp;

but what if you want to cube a number? 

&nbsp;&nbsp;&nbsp;&nbsp;

Well, you can multiply it by itself 3 times...

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   System.out.println(4*4*4); //same thing as 4^3 = 64


```

&nbsp;&nbsp;&nbsp;&nbsp;

But what if you want to raise a number to the 12th power?

&nbsp;&nbsp;&nbsp;&nbsp;

Will you multiply that number by itself 12 times?

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   System.out.println(4 x 4 x 4 x 4 x 4 x 4 x 4 x 4 x 4 x 4 x 4 x 4 x 4); //same thing as 4^12 = 16777216


```
&nbsp;&nbsp;&nbsp;&nbsp;

...yeah no. We don't have all day.

&nbsp;&nbsp;&nbsp;&nbsp;

That's why the math library exists!

&nbsp;&nbsp;&nbsp;&nbsp;

The library comes with a bunch of pre built functions/methods you can use like the pow function (stands for power).

&nbsp;&nbsp;&nbsp;&nbsp;

So, using the power function from the Math library...

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   System.out.println(Math.pow(4, 12)); //same thing as 4^12 = 16777216 but much nicer :)


```

&nbsp;&nbsp;&nbsp;&nbsp;

Note: I'm putting all these Math.methodName()'s inside System.out.println() method so you can copy and paste into your own file and see the result printed for yourself :)

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.pow()

&nbsp;&nbsp;&nbsp;&nbsp;

Raises the first input to the power of the second input.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.pow(5, 2)); //5^2 = 25.0


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Negative Integers

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.pow(-5, 3)); //-5^3 = 125.0


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.pow(2.4,2)); //5.76


```

&nbsp;&nbsp;&nbsp;&nbsp;

Note: The Math.pow method returns only doubles no matter what you put in.

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.min()

&nbsp;&nbsp;&nbsp;&nbsp;

Finds the lowest value out of the two inputs.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.min(1,2)); //1


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.min(1.2,2.4)); //1.2


```

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.max()

&nbsp;&nbsp;&nbsp;&nbsp;

Finds the greatest value out of the two inputs.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.max(1,2)); //2


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.max(1.2,2.4)); //2.4


```

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.abs()

&nbsp;&nbsp;&nbsp;&nbsp;

Takes absolute value of input.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.abs(-1)); //1


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.max(-1.2)); //1.2
   
```
&nbsp;&nbsp;&nbsp;&nbsp;

## Math.sqrt()

&nbsp;&nbsp;&nbsp;&nbsp;

Squares input.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.sqrt(4)); //2.0


```

&nbsp;&nbsp;&nbsp;&nbsp;

Does not work with negative Integers

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.sqrt(-4)); //NaN


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.sqrt(5.76)); //2.4


```

&nbsp;&nbsp;&nbsp;&nbsp;

Note: Always returns doubles no matter what you put in.

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.ceil()

&nbsp;&nbsp;&nbsp;&nbsp;

Rounds whatever you give it to the next whole number (it just rounds up).

&nbsp;&nbsp;&nbsp;&nbsp;

20.9 -> 21

20.2 -> 21 

21 -> 21 (so, there is really no point in providing ints as input for ceil since it will give you the same thing you gave it.)

&nbsp;&nbsp;&nbsp;&nbsp;

Note: Always returns doubles no matter what you put in.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers (no point in this)

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.ceil(21)); //21


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles Ex1

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.ceil(20.9)); //21


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles Ex2

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.ceil(20.2)); //21


```

&nbsp;&nbsp;&nbsp;&nbsp;

## Math.floor()

&nbsp;&nbsp;&nbsp;&nbsp;

Opposite of ceil(). This rounds down- to the FLOOR. Down. Floor. See the connection?

&nbsp;&nbsp;&nbsp;&nbsp;

Note: Always returns doubles no matter what you put in.

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Integers (no point in this)

&nbsp;&nbsp;&nbsp;&nbsp;

```java 

   StdOut.println(Math.ceil(20)); //20


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles Ex1

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.floor(20.2)); //20.0


```

&nbsp;&nbsp;&nbsp;&nbsp;

Works with Doubles Ex2

&nbsp;&nbsp;&nbsp;&nbsp;


```java 

   StdOut.println(Math.floor(20.9)); //20.0


```

### Math.random()

&nbsp;&nbsp;&nbsp;&nbsp;

About:

&nbsp;&nbsp;&nbsp;&nbsp;

<li style = "list-style:none"> (1) Generates a random **double** from 0 to 1.</li>
<li style = "list-style:none"> (2) It always includes the first number. </li>
<li style = "list-style:none"> (3) It does not include the last number unless you add a +1.</li>
<li style = "list-style:none"> (4) If you want it to be a number greater than 0 to 1 then just multiply by the last number. </li>
<li style = "list-style:none"> (5) Inclusive = number should be included. Exclusive = number should be excluded. </li>

&nbsp;&nbsp;&nbsp;&nbsp;

### Example 1 | Generate a random *integer* between 0 and 1 not including 1

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   int randomNum = (int) Math.random() 
   System.out.println(randomNum);


```

&nbsp;&nbsp;&nbsp;&nbsp;


### Example 2 | Generate random *integer* between 1 and 10 (both inclusive)

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   int randomNum = (int) (Math.random() * 10 + 1); 
   // OR int randomNum = (int) (Math.random() * 11); 

   System.out.println(randomNum);


```

&nbsp;&nbsp;&nbsp;&nbsp;

### Example 3 | Generate random *integer* between 0 and 5 (not including 5)

&nbsp;&nbsp;&nbsp;&nbsp;

```java

   int randomNum = (int) (Math.random() * 5 ); 
   System.out.println(randomNum);


```

## Conclusion

&nbsp;&nbsp;&nbsp;&nbsp;

Only Math.pow(), Math.sqrt(), Math.ceil() and Math.floor() return *only* doubles. As in no matter what you put in- in an integer or double- you will *only* get a double back.
